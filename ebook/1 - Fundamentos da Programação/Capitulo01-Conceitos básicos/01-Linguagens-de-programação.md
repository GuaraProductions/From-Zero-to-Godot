# Linguagens de Programação

## O que são Linguagens de Programação?

Para começar a criar jogos na Godot, o primeiro passo é entender que a linguagem de programação funciona como um tradutor entre o que você quer que aconteça e o que o computador realmente faz. Por dentro, o computador é uma máquina que só entende sinais elétricos, que os programadores chamam de binário (aqueles famosos zeros e uns). Como seria impossível para um ser humano escrever um jogo inteiro usando apenas esses números, nós criamos as linguagens de programação. Elas permitem que a gente escreva instruções de um jeito muito mais parecido com o nosso pensamento para criar um software. No fim das contas, tudo o que você vê rodando no PC — desde o Windows e o navegador até a própria Engine da Godot — é um software feito a partir dessas instruções lógicas.

## Níveis de Linguagens

Na história da tecnologia, as linguagens foram evoluindo para ficarem cada vez mais fáceis de usar, o que nos leva aos conceitos de "nível". Existem as linguagens de baixo nível, que são aquelas que falam muito perto das peças do computador (o hardware) e são bem mais difíceis de entender. Já as linguagens de alto nível, como o GDScript que usamos na Godot, são feitas para serem simples e diretas, escondendo toda a parte complicada da máquina para que você possa focar apenas no que o seu jogo precisa fazer. É como se você desse uma ordem "por alto" e o próprio sistema se encarregasse de traduzir os detalhes técnicos para o processador.

## Escolhendo a Linguagem Certa

Cada linguagem de programação é como uma ferramenta diferente em uma caixa de ferramentas: você escolhe a melhor dependendo do que quer construir. Cada uma possui os seus pontos fortes e fracos, e como programador é importante entender que, não existe uma linguagem melhor do que a outra, apenas a linguagem mais ideal para uma determinada tarefa. O mais legal é que, embora existam milhares de linguagens, a base delas é muito parecida. Se você aprender a lógica e o jeito de organizar as ordens (os chamados paradigmas) em uma linguagem, vai conseguir aprender qualquer outra com muito mais facilidade, porque os conceitos fundamentais acabam se repetindo em quase todas elas. Se você não entendeu o que seria esses paradigmas, não se preocupe, apenas continue acompanhando e você vai entender o que eu estou falando.
