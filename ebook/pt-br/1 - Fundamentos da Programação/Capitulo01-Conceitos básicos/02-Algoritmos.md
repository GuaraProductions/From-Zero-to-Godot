# Algoritmos

Antes de começar a digitar qualquer código dentro da Godot, você precisa entender o que é um algoritmo, que nada mais é do que a estratégia ou o planejamento para resolver um problema. No mundo da programação, costumamos dizer que o computador é "burro", porque ele não tem iniciativa própria; ele só possui as ferramentas mais básicas para realizar tarefas e precisa de instruções extremamente detalhadas para cada ação. Um algoritmo, portanto, funciona como uma receita de cozinha ou um guia passo a passo que você dá para a máquina realizar uma tarefa específica.

Para ficar mais claro, imagine que você vai ensinar o computador a fazer um ovo mexido. Você não pode simplesmente dizer "faça um ovo", porque a máquina não sabe o que isso significa. O algoritmo precisaria detalhar cada movimento: pegar os ovos, pegar a frigideira, acender o fogo, quebrar os ovos, colocar os temperos e mexer até ficar pronto. Na programação de um jogo ou de uma calculadora, o processo é idêntico: para somar dois números, o algoritmo dita que primeiro o programa deve ler o número A, depois ler o número B, somar os dois em um novo valor C e, por fim, mostrar o resultado na tela.

## Exemplo Prático de Algoritmo

```
# Algoritmo: Calculadora de Soma

Início 
  Ler o primeiro valor e armazenar na variável A 
  Ler o segundo valor e armazenar na variável B 
  Calcular a soma de A mais B 
  Armazenar o resultado da soma na variável C 
  Exibir (imprimir) o valor de C na tela 
Fim
```

## Regras de um Bom Algoritmo

Existem três regras de ouro para que um algoritmo funcione bem no computador:

1. **Finito**: Ele precisa ter um início, um meio e, obrigatoriamente, um fim; ele não pode rodar para sempre sem concluir o objetivo.
2. **Não ambíguo**: As instruções devem ser precisas e diretas, sem dar margem para o computador escolher o que fazer.
3. **Eficiente**: Um bom algoritmo utiliza os recursos da máquina de forma inteligente para resolver o problema proposto, seja ele simples como uma soma ou complexo como encontrar o melhor caminho em um mapa.
