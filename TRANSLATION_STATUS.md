# Translation Status

This document tracks the translation status for different languages.

## Directory Structure

Each language has its own folder:
- `ebook/{locale}/` - Ebooks in the specified language
- `exercises-lists/{locale}/` - Exercise lists in the specified language
- `introduction/{locale}/` - Introduction README in the specified language

## Contributing Translations

To contribute translations:

1. Copy the `pt-br` folder structure
2. Translate all markdown files
3. Keep the same folder and file structure
4. Submit a pull request

Note: The plugin will automatically show content for the current editor language. If content is not available for a language, a message will be displayed indicating that translations are needed.
